# 📅 Planejamento DEV - SCM

Uma ferramenta completa de gestão ágil (Scrum/Kanban) projetada para desenvolvedores e gestores de projetos. Este sistema une a agilidade do quadro Kanban com a previsibilidade do Gráfico de Gantt em uma única interface.



## 🚀 Funcionalidades Chave
* **Quadro Kanban Interativo:** Sistema de arrastar e soltar (Drag-and-Drop) para movimentação de tarefas entre estados de desenvolvimento.
* **Cronograma (Gantt Chart):** Visualização dinâmica de prazos e duração de tarefas, permitindo identificar gargalos no cronograma.
* **Gestão de Prioridades:** Classificação visual de tarefas por níveis de urgência (Urgente, Alta, Média, Baixa).
* **Filtros Inteligentes:** Pesquisa rápida e filtragem por status para manter o foco no que é prioritário.
* **Persistência de Dados:** Integração com banco de dados em nuvem para armazenamento seguro das sprints e backlogs.

## 🛠️ Stack Tecnológica
* **Frontend:** HTML5 e Tailwind CSS (Design focado em produtividade).
* **Ícones:** Font Awesome 6.
* **Lógica:** JavaScript ES6+ (Manipulação de datas e eventos de arrastar).
* **Estilização Avançada:** Google Fonts (Inter) para legibilidade técnica.

## 🧩 Diferenciais Técnicos
* **Visualização de Gantt Customizada:** Implementação manual de lógica de renderização de cronograma sem dependência de bibliotecas externas pesadas.
* **UX Adaptável:** Interface que alterna entre modos de visualização sem recarregar a página, mantendo o estado da aplicação.
* **Tratamento de Datas:** Sistema de cálculo automático de duração de dias e posicionamento de barras no tempo.

## 📋 Como utilizar
1. Clone o repositório.
2. Configure suas credenciais de banco de dados no script de inicialização.
3. Utilize o botão "Nova Tarefa" para popular seu backlog e arraste para as colunas de execução.
